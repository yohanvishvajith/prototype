# Flask Prototype — setup
## Prerequisites
- Python 3.8+ installed
- Git (optional)
- (Optional) Docker

## Recommended project layout
```
flask_app/
├── app/
│   ├── __init__.py
│   ├── routes.py
│   └── ...
├── tests/
├── requirements.txt
├── .env
└── README.md
```

## Quick setup (cross-platform)
- Windows
python -m venv .venv
.venv\\Scripts\\activate
pip install -r requirements.txt
pip install web3

.env copy rename to .env
python app.py
